Absolute Minimum

In this section, we will be building a minimal Linux system that we can boot using Quick Emulator (QEMU). In order for our system to be called Linux, it needs to run the Linux kernel. The kernel is a core component of the system that interacts with the hardware and manages the resources allocated to processes running on the computer. Without the kernel userspace applications would need to directly interact with the hardware they are running on, which is a difficult and messy task.

Visite kernel.org and click on the yellow button saying "Latest Stable Kernel" to download the Linux kernel source code. Then create a working directory in your home folder named "build" and copy the downloaded tarball into the build folder. Now decompress the tarball (assuming it's called kernel.tar.xz)

$ xz -d kernel.tar.xz
$ tar -xf kernel.tar
$ rm kernel.tar

Remove the tarball, then rename the decompressed folder to "kernel" and change directory into it.

$ cd kernel

Clean the source code folder of any leftovers from previous builds:

$ make mrproper
