Spotted a bug?

If you found an error anywhere in the documentation, feel free to open an issue on this github page. Questions and improvement suggestions can also be submitted there.


Submission Guidelines

  1. Click the green "New Issue" button
  2. Fill in a descriptivep title for the problem
  3. Add any relevant details regarding the problem in the box below
  4. Select appropriate labels for the issue on the right side of the page
    (Press the cog button next to "Labels")
